Bmath Web Application Platform Project (BWAPP) はホームページを作るプラットフォームの開発と、 その上で動く基本的なモジュールを提供します。 また、サイト固有の独自モジュールの開発をサポートするツール群も提供しています。 データベースを使わないサイト構築ツールとしてもご利用頂けます。

開発フレームワークは Zend Framework を、テンプレートエンジンには Smarty を採用しています。

想定するユーザは PHP, HTML, CSS, データベースの知識があり、ある程度サイト構築の経験がある方で 複数のサイトを構築するニーズをお持ちの方です。

このページのカスタマイズについて

このページは webapp/modules/default/templates/index/index.html を読み込んでいます。

カスタマイズする場合は webapp/modules/default/templates_custom というフォルダを作成し そこに編集した index/index.html を保存してください。
